
# Pheedly

A RSS feed reader android application implemented entirely with Jetpack Compose which has some Unit tests, UI tests and also dark theme support.

## Tech Stack

- Jetpack Compose
- Hilt
- MVVM
- Coroutines and Flow
- Navigation Compose
- Room
- Unit Tests with Mockito
- UI Test

## Features

- Light/dark support
- Add rss feeds
- Parse and show articles of RSS
